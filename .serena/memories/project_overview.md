# Project Overview

This is a Next.js-based website for SW Beauty, a beauty salon. The project provides information about salon services, pricing, booking functionality, contact forms, and voucher systems. It includes pages for services, pricing, reservations, contact, about the salon, terms and conditions, privacy policy, and vouchers.

Key features:
- Service listings and categories
- Pricing information
- Online booking system
- Contact forms
- Voucher system
- Instagram feed integration
- Testimonials and highlights
- FAQ section

The website is built for production deployment, likely on Vercel or similar platforms.
