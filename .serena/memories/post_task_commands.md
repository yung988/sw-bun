# Commands to Run After Completing a Task

After making code changes, run these commands to ensure quality:

1. **Format code**: `bun run format`
2. **Lint code**: `bun run lint:fix`
3. **Check all**: `bun run check`

For production builds:
1. **Build**: `bun run build`
2. **Test locally**: `bun run start`

If working with assets:
- **Generate favicon**: `bun run favicon:gen`
